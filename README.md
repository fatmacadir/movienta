# Movienta - OMDB Movie Search Project

Movienta is a modern movie search web application that allows users to quickly explore movies and access detailed information through a simple and user-friendly interface.

Users can search for movies and view detailed information such as:

* Movie posters
* IMDb ratings
* Genres
* Release dates
* Plot summaries

The project uses the OMDB API to fetch movie data dynamically.

The project was designed with a responsive layout to provide a smooth experience on both desktop and mobile devices. It focuses on clean design, usability, and interactive user experience.

## Technologies Used

* HTML
* CSS
* JavaScript
* OMDB API

## Features

* Responsive design
* Dynamic movie search
* User-friendly interface
* API integration
